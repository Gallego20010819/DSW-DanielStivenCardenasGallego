OBJETIVOS DEL SITIO WEB

El objetivo de este sitio web es dar a entender la estructura basica de html5 dando a entender la funcionalidad de cada una de las etiquetas basicas para dar una buena estructuración de una pagina web

Definición de HTML5

Para poder hablar sobre su estructura tenemos que dar a entender que es HTML5

Para la creación de paginas web se necesitan diferentes tipos de lenguajes siendo el html5 uno de ellos dando la estructura principal, siendo este la que organiza el diferente tipo de información que se agrega: Fotos,texto,videos entreo otros que se quieran dar a mostrar en la pagina

Materiales

Imagenes de la funcionalidad de cada una de las etiquetas con su respectiva información

Estructura basica de HTML5

La funcionalidad de cada una de las estructuras de html5 es dar una posisción especifica a la información que se tiene debido a la importancia que se le quiere dar a cada información

Header: Esta es la cabecera de nuestra pagina web, donde podriamos encontrar el nombre de la pagina, slogan y un logo referente al tema que se muestr en cada pagina

Nav: Este es la barra de navegación la cual nos puede mostrar el diferente tipo de información que hay en la pagina web, pudiendo haber un inicio de la pagina, una seccion de información, videos entre otros

Section: esta es la seccion para la información principal de el sitio web, dentro de esta etiqueta conteniendo otra etiqueta llamada article

Article: Esta etiqueta es la que da el contenido de la sección a la que se está ligada utilizando diferentes etiquetas como pueden ser "h1" "h2" siendo el numero mas bajo el de mas importancia o mas llamativo en el sitio web, siendo subtitulos o la información en si de la etiquet section

Aside: Este es la etiqueta que da posición a la información en un lateral, siendo esta información llamada banners o widgets

Footer: esta etiqueta tiene como función dar posición a la información en la parte final del cuerpo de la pagina web, el cual lleva como contenido
